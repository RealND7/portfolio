##  Getting Started

### Prerequisites
- Node.js 20+
- Java 21 (JDK)
- MySQL 8.0+

### Installation

1. **Clone the repository**
   `ash
   git clone <repository-url>
   ` 

2. **Backend Setup**
   `ash
   cd backend
   # Configure application.yml with your MySQL credentials
   ./gradlew bootRun
   ` 

3. **Frontend Setup**
   `ash
   # In the root directory
   npm install
   npm run dev
   `
